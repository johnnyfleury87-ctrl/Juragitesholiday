QUICK START
================================================================================

1. Lire le README:
   $ cat README.md

2. Configurer Supabase:
   - Créer un projet sur supabase.com
   - Exécuter: supabase/schema.sql
   - Exécuter: supabase/seed.sql
   - Créer le bucket "property-photos"

3. Configurer l'app:
   $ cp .env.example .env.local
   # Remplir avec vos clés Supabase

4. Démarrer:
   $ npm install
   $ npm run dev
   # http://localhost:3000

WORKFLOW DE RÉSERVATION
================================================================================

Client:
  1. Browse /logements
  2. Click propriété → /logements/[slug]
  3. Remplir formulaire + Submit
  4. booking_request créé (status: pending)

Admin:
  1. Voir demande en attente via /admin/reservations
  2. Click "Accepter"
     → Crée booking (status: active)
     → Crée payment (status: pending)
     → Update booking_request (status: accepted)
  3. Click "Marquer comme payé"
     → Update payment (status: paid)
     → Crée loyalty_ledger (points = nuits × 10)

Client:
  → Reçoit confirmation
  → Points ajoutés au compte

================================================================================
ADMIN ACCESS
================================================================================

Pour accéder à l'interface admin:

1. Sur la homepage (/)
2. Appuyer sur Ctrl+Shift+A
3. Bouton "Admin" apparaît en bas-droite
4. Cliquer → /admin/login
5. Entrer credentials admin

Note: Ctrl+Shift+A ne casse pas le copier/coller dans les inputs
